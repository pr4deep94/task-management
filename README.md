# Task Manager


## Features

### User-side features

- Login
- Task List
- Add / Edit tasks
- Update tasks
- Delete tasks

## Tools and Technologies

- HTML
- CSS
- Javascript
- Tailwind CSS
- Nest.js
- React
- Mongodb

## To setup it
- Run the below command to install the dependency
npm run install-dependency

- Run the below command to run the project
npm run dev
